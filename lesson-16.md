# Installing Node.js and setting up a local development environment

Always use LTS version for the Production applications.

LTS versions are even numbered.

[Download Node.js](https://nodejs.org/en/download)

We can also use "nvm" to manage multiple Node.js versions on local environment.  
nvm = Node.js Version Manager

[nvm](https://github.com/nvm-sh/nvm/blob/master/README.md)
